# Words Player - 单词卡片学习应用

一个简单的单词卡片学习应用，帮助你记忆和学习单词。

## 功能特点

- 单词卡片展示（正面单词，背面释义和例句）
- 随机打乱单词顺序
- 标记不熟悉的单词
- 导出不熟悉的单词列表
- 学习进度追踪

## 技术栈

- React
- TypeScript
- Vite
- CSS3

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
